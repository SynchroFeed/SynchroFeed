# SynchroFeed Library
*SynchroFeed* is an extensible framework written in Microsoft .NET C# for integrating with a Nuget-like feed. 
The *SynchroFeed* *Library* contains the based functionality, interfaces and plugin extensibility for *SynchroFeed*. 
Most of the functionality of *SynchroFeed* is provided through plugins. This document describes the functionality
within the *Library*.
